Array Problem Set - Lecture 130 & 131

  1. printReverse - takes single array as argument
    • printReverse([1,2,3]) --> 3, 2, 1
    • printReverse([3,6,4,9]) --> 9, 4, 6, 3
  2. isUniform - takes single array as argument
    • isUniform([1,1,1]) --> True
    • isUniform([1,1,1,2]) --> False
  3. sumArray - takes single array as argument
    • sumArray([1,2,3]) --> 6
    • sumArray([10,10,10]) --> 30
  4. max - takes single array as argument
    • max([1,2,3]) --> 3
    • max([1,-5,20]) --> 20
